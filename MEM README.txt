This download includes the R package source folder (MEM), a PDF of the MEM package vignette, and this external README file.

MEM depends on the following packages: gplots, tools, flowCore, grDevices, stats, utils, matrixStats

Install the MEM package from source folder
e.g. > install.packages("C:/Users/You/Documents/MEM_files/MEM",type="source",repos=NULL)
